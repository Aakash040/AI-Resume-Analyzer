## 📌 About the Project

<div align="center">
    <br/><img src="https://raw.githubusercontent.com/Aakash040/AI-Resume-Analyzer/main/screenshots/RESUME.png" alt="screenshot" /><br/><br/>
</div>

This project analyzes resumes using Natural Language Processing (NLP).  
It extracts key information such as skills and keywords, categorizes them, and provides:

- 📊 Resume analysis  
- 🎯 Job role prediction  
- 💡 Skill recommendations  
- 📈 Resume scoring  

---

## 🚀 Scope

- Convert resume data into structured format (CSV)
- Provide insights and analytics for recruiters
- Help users improve resumes through suggestions
- Useful for colleges during placements
- Track trends in job roles and skills
